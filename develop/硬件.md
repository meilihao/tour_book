## FAQ
### 144Hz
1. 显示器连接线支持: 必须使用DisplayPort(推荐), HDMI 2.0+ ,24针的DVI(即支持DVI-D双通道)
1. 设置驱动的刷新率为144Hz

### 检查虚拟机类型
```sh
yum install -y gcc gcc-c++ gdb
wget http://people.redhat.com/~rjones/virt-what/files/virt-what-1.15.tar.gz
tar zxvf virt-what-1.15.tar.gz
cd virt-what-1.15/
./configure
make && make install
virt-what
```
