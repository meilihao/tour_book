# od

## 描述

按(不同进制)输出文件内容

## 参数
- -A 地址进制 : 按指定的进制显示地址信息, 即输出内容的第一列

  进制:
  - d : 十进制
  - o : 八进制(默认)
  - x : 十六进制
  - n :　不打印位移值
- -c : 以字符方式打印文件内容
- -N : 读取限定大小的字节数
- -t 输出格式(允许多个) : 指定数据的显示格式
  
  格式:
    - a : 命名字符(忽略高阶位)
    - c : ascii字符或反斜杠序列(如\n)
    - d : 有符号十进制
    - f : 浮点数
    - o : 八进制(默认)
    - x : 16进制
    - u : 无符号十进制

## 例

    # od -c main.go
