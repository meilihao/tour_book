# 其他

## perf
- perf tools : 综合性能概要分析工具
- ftrace : 追踪kernel的函数调用
- systemtap

## 网络
- nuttcp : 带宽吞吐量测试工具
- netperf : 带宽测试工具
- iperf3 : servers间的带宽测速

	```bash
	# --- 服务端执行
	iperf -s
	# --- 客户端执行
	iperf -c <server_ip>
	```
